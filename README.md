# PubChat
PubChat is a real-time web application designed for seamless public and private communication. Users can participate in global chat rooms, initiate private 1:1 conversations, and personalize their profiles.

## Features
- Public Chat: Join the community and talk to everyone in real-time.

- 1:1 Messaging: Send private invites to users to start dedicated chat threads.

- Invite System: Manage incoming chat requests with a dedicated "Pending Invites" section.

- Profile Customization: Update your profile picture and manage account details.
